# 题解

题意理解如下：**每个方格的数字代表向右/下移动的格数（不能转向）**  
因此有 `f(x,y)=f(x+Map[x][y],y)+f(x,y+Map[x][y])`  
采用记忆化搜索递推即可

const int maxn = 105;
int Map[maxn][maxn];
int ans[maxn][maxn];
bool vis[maxn][maxn];

int n;

int dfs(int x, int y) {
    if (x > n || y > n)
        return 0;

    if (ans[x][y] == -1) {
        ans[x][y] = 0;
        if (x + Map [x][y] <= n)
            ans[x][y] += dfs(x + Map [x][y], y);
        if (y + Map [x][y] <= n)
            ans[x][y] += dfs(x, y + Map [x][y]);
    }

    return ans[x][y];
}
