# 题目

> ## Description  
> > 一个数列的前6项为：1/2，3/5，4/7，6/10，8/13，9/15等，试求数列的第n项(n&lt;2000)  
>    <!--more-->
>   
> ## Input  
> > 输入一个整数n(0< n < 2000)  
>    
>   
> ## Output  
> > 输出数列的第n项  
>    
>   
> ## Sample Input  
> > 1980  
>    
>   
> ## Sample Output  
> > 3203/5183  

# 题解

找规律

 
<b>
对于第n项

分子为前n-1项未出现的自然数中最小的一个

分母为分子+n
</b>

}
